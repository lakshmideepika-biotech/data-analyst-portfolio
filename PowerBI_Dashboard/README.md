# 📊 Power BI Sales Dashboard

## 📌 Project Overview
This project is an interactive Sales Dashboard created using Microsoft Power BI. It analyzes the Sample Superstore dataset to provide insights into sales, profit, quantity, customer segments, and regional performance.

## 🎯 Objectives
- Analyze overall sales performance.
- Identify profitable regions and categories.
- Compare sales across different segments and ship modes.
- Create an interactive dashboard for business decision-making.

## 🛠️ Tools & Skills Used
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Data Visualization
- Data Cleaning
- KPI Cards
- Slicers & Filters
- Interactive Dashboard Design

## 📈 Dashboard Features
- Total Sales
- Total Profit
- Total Quantity
- Sales by Category
- Sales by Sub-Category
- Sales by Segment
- Sales by Ship Mode
- Profit by Region
- Region & Category Filters (Slicers)
